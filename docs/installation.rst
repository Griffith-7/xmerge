Installation
============

Requirements
------------

- Python 3.10+
- PyTorch 2.0+
- Transformers 4.30+
- Datasets 2.0+

Install from PyPI (once published)
-----------------------------------

.. code-block:: bash

    pip install xmerge

Install from GitHub
-------------------

.. code-block:: bash

    pip install git+https://github.com/Griffith-7/xmerge.git

Editable install (for development)
-----------------------------------

.. code-block:: bash

    git clone https://github.com/Griffith-7/xmerge.git
    cd xmerge
    pip install -e ".[dev]"
